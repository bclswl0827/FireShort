# FireShort

FireShort：基于 Firebase 的 Serveless 纯前端短链接生成工具

本项目不依赖后端，仅在前端使用 JS 跳转到目标网址，且 URL 结构可同普通短网址程序一样。

项目说明可以参考 [我写了一个无需后端的短网址程序... ](https://ibcl.us/ShortLink-Firebase_20230626/)。
